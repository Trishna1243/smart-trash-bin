# Smart Trash Bin

This project uses YOLOv5 to detect garbage from videos.

## Steps
- Dataset imported from Roboflow
- Model trained using YOLOv5
- Detection performed on video input

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run detection:
python detect_video.py

## Output
- Video with bounding boxes
